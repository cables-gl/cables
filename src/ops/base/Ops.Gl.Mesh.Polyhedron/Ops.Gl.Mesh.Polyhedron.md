
Generate and modify polyhedron meshes to create complex topologies.


See this page for more information about the notation:
https://github.com/mikolalysenko/conway-hart

Seeds:
 - "T" Tetrahedron
 - "C" Cube
 - "O" Octahedron
 - "D" Dodecahedron
 - "I" Icosahedron
 - "Pn" Prism
 - "An" Antiprism
 - "Yn" Pyramid

Operations:

 - a Ambo
 - b Bevel
 - c Canonicalize
 - d Dual
 - e Expand
 - g Gyro
 - j Join
 - kn Kis
 - m Meta
 - o Ortho
 - p Propellor
 - r Reflect
 - s Split
 - tn Truncate
